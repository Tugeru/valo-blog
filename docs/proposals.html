Radiant Intel

Architectural Proposals

1. Pure Blade Monolith

Standard Laravel MVC with zero extra JS dependencies. Solid, fast, and easy to maintain.

  • Bespoke AuthControllers
  • Blade Components for UI
  • Standard Form POSTs
  • Native Browser Confirms
Best for: Extreme simplicity & stability.

2. Blade + Alpine.js Recommended

Blade for structure, Alpine.js for "tactical" UI micro-interactions and seamless image previews.

  • Instant Image Upload Previews
  • Animated Delete Modals
  • Interactive Hover States
  • Bespoke Auth Controllers
Best for: The full "Valorant Client" feel.

3. Admin Livewire SPA

Uses Laravel Livewire for a reactive, Single Page Application feel in the Admin Dashboard.

  • No page refreshes in Admin
  • Real-time validation
  • Search/Filter posts instantly
  • Bespoke Auth Controllers
Best for: High-volume content management.

Selected Stack: Laravel 12 + Tailwind 4 + SQLite
